# CREATE

## DB 생성

```sql
CREATE DATABASE <db_name>;

CREATE DATABASE test_dbdb;
```

## Table 생성

```sql
CREATE TABLE <name> (<column_name> <data_type>, <column_name> <data_type>);

CREATE TABLE test_table (name char(20), age int);
```

<br>

# 테이블 구조 보기

```sql
DESCRIBE <table_name>
```

# DROP

```sql
DROP DATABASE <name>
DROP TABLE <name>
```
